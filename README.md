# Iron Green — Projeto Final (Desenvolvimento de Jogos)

Descrição
---------
Iron Green é o projeto final da disciplina de Desenvolvimento de Jogos. Este repositório contém a versão distribuível do jogo em formato executável (.exe). O objetivo deste README é explicar de forma simples como executar o jogo.

Como executar
-------------
1. Abra um terminal (PowerShell, Prompt de Comando) ou faça via interface gráfica.
2. Clone o repositório:
   ```
   git clone https://github.com/reimbazz/Iron-Green.git
   ```
3. Entre na pasta do projeto:
   ```
   cd Iron-Green
   ```
4. Localize o arquivo executável (.exe) na raiz do repositório ou na pasta `dist/` (ou outra pasta de distribuição).
   - Exemplo de execução pelo terminal:
     - PowerShell:
       ```
       .\NomeDoExecutavel.exe
       ```
     - Prompt de Comando:
       ```
       NomeDoExecutavel.exe
       ```
   - Ou dê duplo clique sobre o arquivo `.exe` no Explorador de Arquivos do Windows.

Observações
----------
- Substitua `NomeDoExecutavel.exe` pelo nome real do arquivo presente no repositório.
- Ao baixar um executável da internet, o Windows pode bloquear o arquivo. Para desbloquear: clique com o botão direito → Propriedades → marque "Desbloquear" se essa opção aparecer.
- Se o jogo não iniciar e aparecer erro de DLL ausente (ex.: MSVCP140.dll), instale o Visual C++ Redistributable (versão 2015–2022) da Microsoft.

Compilar a partir do código-fonte
---------------------------------
Se o repositório incluir o código-fonte e você quiser compilar localmente, verifique a pasta `Source/` ou a documentação presente no repositório para instruções específicas (por exemplo, qual engine/versão usar — Unity, Godot, Unreal, etc.). Caso não haja instruções, informe qual engine foi usada para que possamos adicionar um guia de build.
